hexoをつかってGithubページにブログを作る

HEXOという静的ファイルジェネレータとGithubページが相性が良いので、HEXOでブログみたいなことをヤってみようと思う。

前提条件

  • PCはMac
  • NodeJSインストール済み
  • npmインストール済み

githubページを用意する

githubにログインし、ヘッダの＋ボタンから、新規リポジトリを作成する。

Repository nameには、URLになる名前を入力する。
必ず最後は@github.ioにすること。

例：myblog@github.io

ただし、サブドメインと同じ扱いなので、すでに使われているものは使えない。

HEXOでページを作るので、README.mdは自動生成させない。
つまり、Initialize this repository with a READMEはチェックしないでおく。

プラグイン

すこし便利に使いたいので、以下の2つのプラグインも入れておく。

  • hexo-deployer-gitは、githubページヘ一発でデプロイしてくれるプラグイン
  • hexo-generator-feedは、rssフィード(atom.xmlファイル)を自動生成してくれるプラグイン

コンフィグ

ブログのタイトル、作者などの設定は、_config.ymlで行う。
エディタで開いて適宜編集しておく。

画像を使う場合も想定して、以下の項目をtrueにしておく。
post_asset_folder: false
↓
post_asset_folder: true

一番下にdeployという項目があるので、ここをgithubページのURLにしておく。

※設定ファイルってのはだいたいそんなもの(起動時にだけメモリに読み込まれる一方通行)。

ローカルサーバを止めるのは、メッセージにある通り、Ctrl+Cもしくはcmd+.でOK。

テンプレートのデザインを変える

幾つかのテンプレートが用意されているが、ほとんどが中華製なので、テーマにある_config.ymlが中国語だったりする。

しかし、Google翻訳を使いながら、自分流にカスタマイズすれば問題ない。

  1. Githubのテーマから好きなリポジトリをcloneする
  2. テーマ内にある_config.ymlを適宜編集する
